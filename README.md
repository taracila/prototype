### Setup

Run the following commands in the Terminal below to prepare and run the sample app:

1. `cd ~/workspace/gulp-sample-app`
2. `npm install gulp -g`
3. `gem install sass`
4. `npm install`
5. `gulp`

Go to the "Preview Menu" and click "Port 4000"
